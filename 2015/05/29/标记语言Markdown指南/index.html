标记语言Markdown指南

发表于 2015-05-29   |   分类于 Hexo   |  

Markdown是一种轻量级的标记语言，通过一定的语法，它可以使普通的文本文件具有一定的格式。其目标是易写易读，语法非常简单，学习容易，一旦熟能生巧之后，会有一劳永逸的效果。

为什么要学习Markdown语言

正如前面所说，Markdown是一款轻量级标记语言，它用简洁的语法代替了排版，使我们不再烦恼于word或pages那些繁琐的字体格式设置。从内容到格式，从插图到链接，都可以通过键盘输入，从而使我们专注于写作。写作完成的Markdown文章可以导出为HTML格式发布到网上，也可以通过衍生功能导出更多的格式，如LaTex,PDF等。如果你有一份使用Markdown制作的线上&线下简历，HR一定会对你好感大增。目前Markdown得到了来自于Ruby和Github社区的大力支持，很多网站（如简书），世界上最流行的博客平台WordPress和大型CMS如Joomla、Drupal都能很好的支持Markdown。本文即是使用Markdown完成。

Markdown的优点

  • 基于纯文本格式，可以用任何文本编辑器编辑。相比纯文本格式拥有更丰富的格式，可以插入图片链接。
  • 相对于word而言，使用Markdown进行写作，让你更专注于你的内容，而不会为排版分心。采用Markdown编辑的文件可读可写，而word文档中含有很多格式数据，只能借助少数工具读取。
  • 学习成本低廉，常用语法规则非常少，记忆负担小。如果你接触过html语言，那么学习成本接近于零。
  • 语言拓展性强，通过各种衍生版本可以实现各种拓展功能。
  • 可以导出多种格式的文件，主流网站的支持度好。

基本语法

标题

代码：

# 一级标题
## 二级标题
...
###### 最多六级标题

效果：

如前面所示，只要在行首加若干#就可以代表不同等级的标题，Markdown最多提供了六级标题设置。

列表

代码：

- 无序列表
- 无序列表
- 无序列表

1. 有序列表
2. 有序列表
3. 层级列表

- 分级列表
    + 分级列表
    + 分级列表
- 分级列表

效果：

  • 无序列表
  • 无序列表
  • 无序列表
  1. 有序列表
  2. 有序列表
  3. 层级列表
  • 分级列表
    • 分级列表
    • 分级列表
  • 分级列表

在Markdown中只要在行首加入-，1.就可以分别实现无序列表和有序列表。符号和文字之间要要存在一个空格。

引用

代码：

> 天行健，君子以自强不息。

效果：

天行健，君子以自强不息。

在Markdown中，只需要在文字前加入>就可以实现引用的效果。

代码块

代码：

var i = "Hello, ";
var j = "World!";
echo i.j;

效果：

var i = "Hello, ";
var j = "World!";
echo i.j;

作为程序员，代码框对于我们来说时必不可少的工具，在Markdown中代码块十分容易实现。上面效果不是很明显，代码块的实际语法是在行首加入四个空格，那么此行的内容就都会出现在代码块中。

粗体和斜体

代码：

**这是粗体**

*这是斜体*

效果：

这是粗体

这是斜体

Markdown中的粗体和斜体的实现非常简单，当使用将**文字包围在中间时，文字就会变成粗体，当使用*将文字包围在中间时，文字就会变成斜体。

空格，换行，分割线

代码：

一个回车
视为空格

两个以上的回车

才能分段

行尾两个空格  
也可以换行

这代表分割线
---

效果：

一个回车
视为空格

两个以上的回车

才能分段

两个空格
可以换行

这代表分割线

链接与图片

代码：

[这是超链接](http://www.wanghao365.com)

[!这是图片链接](/img/WangHao++.png)

效果：

这是超链接

图片与链接的格式十分相似，只需要在前面加入一个!就可以实现。

直接插入HTML

在Markdown文件中，可以在新的段落直接插入html语句，这对于熟悉html的程序员们真是超级方便。当觉得Markdown提供的格式已经不能满足你的需求的时候，直接书写html来实现想要的效果，这使得markdown文件具有了各种可能性。

小结

以上就是Markdown的基本介绍，提供的不到10个基本语法已经可以满足日常书写的基本需要了。Markdown虽然简单，但终究是一门语言，想要随心所欲的应用还是需要多加练习。当你能够熟练掌握的时候，一定能体会到专注写东西的乐趣。
